Octree Color Quantization

12/05/2022

A color matrioska.

Demo

To get an idea of how this work. Imagine an Octree as an XYZ axes plot. We can insert an RGB color by using each channel as a position. The r channel is the x coordinate, g is y and b is z.

Then you have to pick the smallest possible sub-cube that contains the number of colors specified in the quantization function.
